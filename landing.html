Learning Outcomes

I inquired into the diversity of different data types, data sources, data storage format and methods of data collection.

Next, I explored the concepts of big data, big data security, big data security challenges and how to handle big data security challenges.

Later on, I grew my comprehension of the process of data wrangling and the challenges faced during data wrangling. I discovered how to use data wrangling to eliminate duplications and redundancies, thereby improving the quality of data collected in the raw state to a more usable format through cleansing and normalization.

I am now well acquainted with the methodologies and tools in data wrangling.
