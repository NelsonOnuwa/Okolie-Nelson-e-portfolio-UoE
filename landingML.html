Machine Learning (ML)

This module introduces an understanding of the most widely-used machine learning and deep learning methods on different data sets with the use of Jupyter notebooks.

Machine learning techniques were applied to Airbnb dataset to measure how well a model can correctly predict prices.

Deep learning algorithm were applied to CIFAR-10 dataset to measure how well a model can classify images .

The legal, social, ethical and professional issues machine learning professionals would likely experience were revealed in this module.

During the team project in this module, effective team member skills in a virtual professional environment was developed and encouraged .
